# 投资与宏观基础学习路线 README

你现在卡住的点，本质上不是“记不住新闻”，而是还没有把下面这条链条串起来：

**政策工具 → 短端利率预期 → 美债收益率曲线 → 企业估值/融资成本 → 风格轮动 → 股市表现**

学完这份路线，你应该能做到三件事：

1. 看懂美联储声明、议息会议、点阵图、缩表/扩表在讲什么。
2. 看懂 2 年期、10 年期美债收益率为什么会动，以及它和股市的关系。
3. 遇到新闻时，能自己判断它更像是“短期情绪扰动”还是“改变估值中枢的东西”。

---

## 0. 先建立一个总框架

你看到的这些东西，分别属于不同学科：

* **宏观经济学**：通胀、失业、GDP、增长、衰退、货币政策、财政政策。
* **货币金融学 / 中央银行学**：美联储如何通过利率、公开市场操作、资产负债表影响市场。
* **固定收益 / 债券市场**：国债价格、收益率、期限结构、久期、久期风险。
* **公司金融 / 估值**：折现率、现金流、无风险利率、风险溢价、PE/DCF。
* **市场微观结构与行为**：预期、交易、流动性、风险偏好、拥挤交易。

你现在需要的不是“先把所有学科学完”，而是先抓住**传导链条**。

## 2. 最重要的传导链条

你先把下面这条链条背熟，再去看新闻：

### 情景 A：美联储降息预期升温

1. 市场开始预计未来短端利率会下降。
2. 2 年期美债收益率通常更敏感，因为它更贴近未来政策路径。
3. 债券收益率下行，债券价格上行。
4. 折现率下降，股票的理论估值上升。
5. 高成长股通常更受益，因为它们的现金流更远、对折现率更敏感。

### 情景 B：美联储说要继续加息/更久维持高利率

1. 市场重新上调未来短端利率预期。
2. 2 年期收益率往往先动，10 年期也会受影响。
3. 融资成本上升，企业借钱更贵，消费和投资可能变弱。
4. 估值折现率上升，股票估值中枢下移。
5. 依赖远期现金流的成长股通常更受压。

### 情景 C：缩表

1. 美联储减少资产持有量，市场流动性边际收紧。
2. 债券供给/需求结构变化，长期收益率可能上行。
3. 金融条件收紧，风险资产承压。

### 情景 D：扩表

1. 美联储买入资产，流动性改善。
2. 长端利率可能下降或至少被压住。
3. 风险偏好提升，股票、信用债、成长资产可能受益。

## 3. 先学什么，再学什么

### 第 1 阶段：看懂新闻的“词汇表”

目标：你看到新闻时，不再只知道“好像很重要”，而是知道它在描述哪一层。

你要掌握：

* FOMC、点阵图、利率决议、新闻发布会
* CPI、PCE、核心 PCE、非农、失业率
* 2 年期、10 年期国债收益率
* QT、QE、资产负债表
* 风险偏好、避险、久期、收益率曲线

### 第 2 阶段：理解“政策如何影响市场”

目标：知道为什么政策变化会传导到股票和债券。

你要掌握：

* 央行如何影响短端利率
* 短端利率如何影响整个收益率曲线
* 收益率曲线如何影响公司融资和估值
* 利率为什么会影响成长股、银行、地产、周期股

### 第 3 阶段：理解“估值为什么会变”

目标：你能自己从折现率角度解释估值波动。

你要掌握：

* DCF 的基本逻辑
* 无风险利率 + 风险溢价 = 折现率的一部分
* 为什么高利率环境下，久期长的资产更难涨

### 第 4 阶段：建立自己的跟踪框架

目标：每周只看少量关键指标，但看得明白。

你要能跟踪：

* FOMC 会议结果
* CPI / PCE
* 2Y、10Y 美债收益率
* 收益率曲线是否倒挂
* 市场对降息/加息的预期变化
* 你的持仓属于哪类利率敏感资产

## 6. 你应该怎样读新闻

每次看到一条新闻，你都问自己 4 个问题：

1. 这是在讲**政策预期**，还是在讲**真实经济数据**？
2. 影响的是**短端利率**，还是**长端利率**？
3. 影响的是**估值中枢**，还是**短期情绪**？
4. 对我的持仓来说，是利多、利空，还是其实没那么重要？

举例：

* “美联储偏鹰” → 未来利率可能更高 → 折现率上升 → 估值承压。
* “10 年期美债收益率上升” → 长端折现率上升 → 成长股压力通常更大。
* “缩表加速” → 流动性更紧 → 风险资产可能更难受。

## 7. 做笔记的方法

不要做“抄新闻式笔记”，要做“因果链笔记”。

每条笔记只写 4 行：

* 发生了什么
* 市场在交易什么预期
* 对利率和收益率曲线的影响
* 对股市/板块/持仓的影响

示例：

```text
事件：FOMC 表态更鹰。
市场预期：降息更晚，政策利率维持更久。
利率影响：2Y 先上行，长端利率也可能被带动。
资产影响：高估值成长股承压，现金流稳定板块相对占优。
```

## 11. 中国市场（A 股）和全球市场怎么一起理解

你现在做的是 A 股，但你会发现：

* 美国降息预期会影响全球风险偏好。
* 美债收益率会影响全球资金定价。
* 美元指数会影响人民币汇率与北向资金。
* 中国央行、财政、产业政策又会单独影响 A 股。

所以你其实是在看：

**全球流动性 + 中国政策 + 行业周期 + 市场情绪**

这份学习路线是可以覆盖的，但你需要把“美国链条”和“中国链条”并列理解。

---

### 美国链条（全球定价核心）

核心变量：

* 美联储利率
* 美债收益率
* 美元指数
* 全球风险偏好

影响路径：

```text
美联储偏鹰
→ 美债收益率上升
→ 全球风险资产估值受压
→ 北向资金可能流出
→ A 股高估值板块承压
```

特别是：

* AI/科技/新能源/高估值成长股
* 港股互联网
* 高外资持仓板块

通常会更敏感。

---

### 中国链条（A 股更直接）

核心变量：

* 人民银行政策（MLF、LPR、降准、逆回购）
* 财政刺激
* 地产政策
* 产业政策
* 国务院/证监会表态
* 社融、信贷、PMI

影响路径：

```text
中国降准/降息
→ 银行体系流动性改善
→ 企业融资成本下降
→ 风险偏好改善
→ A 股估值可能修复
```

或者：

```text
地产政策放松
→ 地产链预期改善
→ 银行/建材/家居/消费链修复
→ 指数权重改善
```

---

### 你以后应该建立“双层框架”

以后每次看新闻，都先问：

#### 第一层：全球流动性有没有变

主要看：

* 美联储
* 美债收益率
* 美元指数
* 全球风险偏好

#### 第二层：中国内部政策有没有变

主要看：

* 央行
* 财政
* 地产
* 产业政策
* 监管

最后再落到：

```text
对 A 股哪个板块影响最大？
```

## 14. 你怎么确认自己的理解是不是对的

这是最关键的问题。

因为宏观最容易出现：

```text
“我以为自己懂了，但其实只是看了很多新闻。”
```

真正的学习，不是“知道新闻”，而是：

```text
能提前猜市场会怎么看。
```

## 15. 校验自己理解的方法（最重要）

### 方法 1：先预测，再看市场反应

这是最重要的方法。

每次重大数据发布前：

你先写：

```text
如果 CPI 高于预期：
- 市场会不会认为降息更晚？
- 2Y 会不会上升？
- 成长股会不会承压？
```

然后再看市场实际怎么走。

长期下来，你会建立“市场语言”。

---

### 方法 2：观察“哪个资产先动”

宏观里有个重要思想：

```text
不同资产在交易不同东西。
```

例如：

* 2Y：交易未来政策利率。
* 10Y：交易长期增长、通胀、期限溢价。
* 黄金：交易实际利率与避险。
* 美元：交易全球流动性与相对利差。
* 纳指：交易流动性与成长估值。

所以：

不要只看股市。

要看：

```text
“谁先反应？”
```

---

### 方法 3：复盘“为什么市场和新闻反着走”

这是进步最快的方法。

例如：

```text
明明降息，为什么股市跌？
```

因为：

可能市场之前已经提前交易了；
或者市场认为：

```text
“降息是因为经济太差。”
```

所以：

宏观最核心的一句话是：

```text
市场交易的是“预期差”，不是新闻本身。
```

---

### 方法 4：做“因果链笔记”而不是新闻摘抄

错误笔记：

```text
今天美联储讲话偏鹰。
```

这种笔记几个月后毫无意义。

正确笔记：

```text
事件：鲍威尔讲话偏鹰。

市场预期：
市场认为降息可能推迟。

利率影响：
2Y 上升，收益率曲线重新定价。

资产影响：
成长股承压，银行相对受益。
```

然后你第二天去验证：

```text
2Y 是否真的上升？
纳指是否真的弱？
银行指数是否相对更强？
```

你会越来越接近真正理解。

## 16. 一个非常重要的认知

你以后会发现：

真正厉害的投资者，
不是知道更多新闻的人。

而是：

```text
能把新闻翻译成“资产定价语言”的人。
```

例如：

```text
CPI 高于预期
≠ 只是一个新闻

而是：
→ 降息预期下降
→ 短端利率上升
→ 折现率提高
→ 高估值资产承压
```

这才是你真正要训练的能力。

## 17. 最后的建议

不要急着：

* 预测指数
* 猜顶猜底
* 每天追热点

先把：

```text
宏观 → 利率 → 流动性 → 估值 → 板块
```

这条链条建立起来。

当你能稳定理解这条链条后：

新闻会从“噪音”变成“结构化信息”。
